# ASP.NET Core Identity Security Project

## Project Overview
This project demonstrates authentication and authorization mechanisms using ASP.NET Core Identity.

## Technologies Used
- ASP.NET Core
- Entity Framework Core
- Identity
- SQL Server
- Bootstrap

## Features
- User Registration
- Login / Logout
- Email Confirmation
- Role-Based Authorization
- Password Reset
- Identity Security Configuration
